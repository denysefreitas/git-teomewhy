Esse arquivo contém os comandos secundários do git.

git diff: usados para saber quais foram as alterações específicas;
git reset <arquivo>: altera o estado do arquivo (ctrl + z do git add);

git checkout -b <nova_branch>: cria uma nova branch e altera para a que foi criada;
git branch: exibe a branch atual;
git checkout <branch>: altera a branch atual;
